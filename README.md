C# Sentiment Analysis (Hugging Face API Integration)

A .NET console application developed in C# that performs real-time sentiment analysis on user-provided text. The project utilizes Hugging Face's Inference API to communicate with the advanced NLP model.

Key Features
- Advanced NLP Integration
- Asynchronous Architecture
- Secure Key Management
- Dynamic Response Parsing

Tech Stack
- Language: C#
- Framework: .NET Core / .NET 8
- AI Models & API: Hugging Face Inference API 
- Libraries: System.Text.Json, System.Net.Http
